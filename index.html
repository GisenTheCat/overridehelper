To set up overrides, you need to open DevTools (Ctrl + Shift + I) and open the Sources tab If you can't find it, please press the >> sign to the right Next, go to the Overrides tab which is located underneath the menu you just used Click the "Select folder for overrides" button, make a folder anywhere and name it something like "overrides" Select it and press Open Next press the button below and select the folder you used Only the button has to be pressed if you use this website later DevTools has to be open when you press the button
